# Audit de sécurité complet d'une PME

## Contexte du projet

**Entreprise** : PME industrielle de 75 employés (fabrication de pièces électroniques)

**Infrastructure IT** :
- 3 serveurs Windows Server 2019 (AD, fichiers, ERP)
- 1 serveur Linux (site web vitrine)
- 70 postes Windows 10
- Firewall Fortinet FortiGate 60E
- Switch HP réseau
- 2 connexions Internet (fibre 500 Mbps + backup ADSL)

**Déclencheur** : Assurance cyber demande un audit de sécurité avant renouvellement de la police (exigence suite à plusieurs sinistres dans le secteur)

**Objectif** : Audit complet de la sécurité IT, identification des vulnérabilités critiques et plan de remédiation dans les 4 semaines.

**Budget** : 8 000€ (audit + corrections prioritaires)

**Mon rôle** : Consultant en cybersécurité, mission de 3 semaines
